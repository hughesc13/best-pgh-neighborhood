# Neighborhood Analysis

A tool that scores Pittsburgh neighborhoods based on the availability of key public amenities such as playgrounds, pools, and shelters.

## 📊 Datasets Used
All datasets were obtained from the [Western Pennsylvania Regional Data Center (WPRDC)](https://data.wprdc.org/):

- [City of Pittsburgh Playgrounds](https://data.wprdc.org/dataset/city-of-pittsburgh-playgrounds)
- [City of Pittsburgh Pools](https://data.wprdc.org/dataset/city-of-pittsburgh-pools)
- [City of Pittsburgh Shelters](https://data.wprdc.org/dataset/city-of-pittsburgh-shelters)


## 🔎 Overview
This repository calculates and visualizes a weighted amenities score for Pittsburgh neighborhoods to help identify the most well-resourced areas based on publicly available recreational infrastructure.
